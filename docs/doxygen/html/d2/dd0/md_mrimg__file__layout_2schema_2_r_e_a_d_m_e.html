Macrium Reflect X - JSON Schema

This folder contains 2 JSON schema versions:

  1. complete_schema.json - This is a comprehensive schema for every JSON node in the backup file JSON data. This includes auxiliary data, including the backup environment and the settings used to create the backup.

  2. required_schema.json - This simplified schema only contains JSON nodes that are required for reading and recovering backup files.

‍[!TIP] JSON schemas can be easily navigated using the Atlassian JSON Schema viewer. See below:
