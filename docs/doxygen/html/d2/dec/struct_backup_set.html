Detailed Description

Represents a set of backup files and their associated handles.

This structure contains a vector of unique pointers to BackupFileAndHandle objects, which represent backup files and their associated handles. It also contains a map that maps an integer index to a std::fstream*. This map can be used to quickly look up a std::fstream* based on its associated index.

std::fstream * BackupSet::getFileHandle ( const int index)

Retrieves the file handle for a given file number in a backup set.

This function iterates over all the file handles in the backup set. If the file number of the current file handle matches the requested index, the handle is returned. If no matching file handle is found, an exception is thrown.

Parameters
indexThe file number to search for.
Returns
The file handle for the file with the given number.
Exceptions
std::runtime_errorif no file with the given number is found in the backup set.
