Detailed Description

This file defines a set of wrapper functions for encryption and decryption operations using the OpenSSL library. These functions provide a higher-level, more convenient interface for performing common cryptographic operations such as AES encryption/decryption, HMAC computation, and MD5 hash computation. They also handle error checking and throw exceptions if any of the operations fail, simplifying error handling in the code that uses these functions.

Initializes an AES operation (encryption or decryption) based on the provided parameters.

This function initializes an AES operation (either encryption or decryption) using the provided parameters. It creates and returns a unique_ptr to an initialized EVP_CIPHER_CTX, which will automatically be freed when it goes out of scope.

Parameters
aes_variantThe AES variant to use (10 for AES-128, 12 for AES-192, 14 for AES-256).
pDerivedKeyThe derived key to use for the operation.
pIVThe initialization vector to use for the operation. Can be NULL for ECB mode.
TypeIf set to EncryptionOperation::Encrypt, the function initializes for encryption. If set to anything else, it initializes for decryption.
Returns
A unique_ptr to the initialized EVP_CIPHER_CTX. The unique_ptr will automatically free the context when it goes out of scope.
Exceptions
std::runtime_errorIf an error occurs during initialization.
