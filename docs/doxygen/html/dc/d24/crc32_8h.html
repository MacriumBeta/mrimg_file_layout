Calculates the CRC32 checksum for a buffer of data.

This function calculates the CRC32 checksum for a given buffer of data. It uses the polynomial 0x04C11DB7. The function can also continue a previous CRC calculation by providing the previous CRC value as the last parameter.

Parameters
bufThe buffer containing the data to calculate the CRC for.
lenThe length of the data in the buffer to calculate the CRC for.
BufferlenThe total length of the buffer.
ulCRCThe initial CRC value. This can be the result of a previous CRC calculation to continue from. Defaults to 0xffffffff.
Returns
The calculated CRC32 checksum.
