uint32_t initialCRC )

Calculates the CRC32 checksum for a buffer of data.

This function calculates the CRC32 checksum for a given buffer of data. It uses the polynomial 0x04c11db7. The function can also continue a previous CRC calculation by providing the previous CRC value as the last parameter.

Parameters
bufferThe buffer containing the data to calculate the CRC for.
dataLengthThe length of the data in the buffer to calculate the CRC for.
bufferLengthThe total length of the buffer.
initialCRCThe initial CRC value. This can be the result of a previous CRC calculation to continue from. Defaults to 0xffffffff.
Returns
The calculated CRC32 checksum.

◆ Initialize_CRC32_Table()

void Initialize_CRC32_Table ( )

Initializes the CRC32 lookup table.

This function generates a lookup table for the CRC32 calculation. It should be called once before any CRC32 calculations are done. The lookup table uses the polynomial 0x04c11db7, which is the official polynomial used by CRC-32 in PKZip, WinZip and Ethernet.
