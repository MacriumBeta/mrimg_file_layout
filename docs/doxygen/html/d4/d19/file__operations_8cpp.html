◆ openFileWithGuard()

SharedFile openFileWithGuard ( const std::wstring & filename)

Opens a file and returns a shared_ptr to the file stream with a custom deleter.

This function opens a file with the given filename and returns a shared_ptr to the file stream. The shared_ptr has a custom deleter that automatically closes the file when the shared_ptr is destroyed. This ensures that the file is always closed properly, even if an exception is thrown.

Parameters
filenameThe name of the file to open.
Returns
A shared_ptr to the file stream, with a custom deleter that closes the file.
