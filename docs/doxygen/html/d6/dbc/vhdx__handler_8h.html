Handles the creation and mounting of the VHDX file.

This function gets the disk layout for the disk to be restored, constructs the VHDX file name, creates the VHDX file, and mounts it.

Parameters
filenameThe name of the backup file.
outputPathThe path where the output should be written (optional).
vhdxNameA reference to a string where the VHDX file name will be stored.
backupFileThe backupFile object that holds the parsed JSON data from the backup file.
diskNumberThe number of the disk to be restored (-1 means the first disk in the backup file).
Returns
The VHDXManager object that manages the VHDX file.
