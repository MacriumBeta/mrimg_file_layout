8This file defines a set of wrapper functions for encryption and decryption
9operations using the OpenSSL library. These functions provide a higher-level,
10more convenient interface for performing common cryptographic operations such
11as AES encryption/decryption, HMAC computation, and MD5 hash computation.
12They also handle error checking and throw exceptions if any of the operations
13fail, simplifying error handling in the code that uses these functions.
